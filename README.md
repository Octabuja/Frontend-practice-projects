Frontend Practice Projects

Colección de proyectos desarrollados durante mi proceso de aprendizaje de desarrollo web frontend.

El objetivo de este repositorio es practicar conceptos fundamentales de HTML, CSS y JavaScript mediante la construcción de aplicaciones y componentes web de pequeña escala.

Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Git & GitHub

Proyectos incluidos

* Landing Pages
* Formularios interactivos
* Calculadoras
* Componentes UI
* Ejercicios de manipulación del DOM
* Prácticas de diseño responsive

Objetivos de aprendizaje

* Estructuración de páginas con HTML
* Estilos y maquetación con CSS
* Manipulación del DOM con JavaScript
* Eventos y validaciones
* Diseño responsive
* Buenas prácticas de desarrollo web

Estado del repositorio

Repositorio destinado a prácticas y ejercicios de aprendizaje frontend.

Los proyectos aquí contenidos representan distintas etapas de formación y experimentación con tecnologías web.
